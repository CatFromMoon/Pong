ООП в коде проекта можно взять наследование класов, например:
class Ball : public Middle (Ball.h)
Благодаря чему можно получить доступ к 
protected:
    void notify(int  a, Mix_Music* sound)

Полиморфизм (виртуальная функция):
virtual void onNotify(int a, Mix_Music* sound) = 0;

Инкапсуляция (ограничение доступа):
protected:
	void notify(int  a, Mix_Music* sound)
	{
		b->onNotify(a, sound);
	}

private:
	Music aa;
    Observer* b = &aa;
};

Функции имеют краткое описание. Весь код разбит по файлам
название которых соответствует действиям которые делают эти функции.
Весь код придержан однородной стилистике.


При написании использовано внешние библиотеки для текстур(SDL2_image),
треков(SDL2_mixer) и текста(SDL2_ttf)


В качестве архитектури можно взять стандартный пример когда отрисовка 
происходит после того, как было обработано нажатие клавиш и изменение 
в позиции объектов. В данном случае обработка изменений позиции шара 
относительно двух игроков может быть использована отдельно в других 
проектах где также нужно будет обрабатывать тоже самое что позволяет 
использовать её так, как будто би это движок для обработки шара.


В проекте используется патерн Observer для наблюдения за шаром. 
Когда шар ударяется в игрока или вилетает за границу происходит 
запуск того или иного трека Реализация в файлах Observer.h, 
Music.h, Ball.h


Код у Visual Studio 2019 компилировался как х86 (32 битная разрядность).
Библиотеки SDL2 также 32 разрядные. Запустить можно как проект 
Visual Studio или просто с папки файлом Pong.exe. На случай отсутствия 
msvcp140d.dll, ucrtbased.dll или vcruntime140d.dll при запуске Pong.exe 
их нужно копировать из папки DLL_2015 в папку с Pong.exe.


------ПРИ ЗАПУСКЕ ИГРЫ------
Откроется окно которое предложит нажать 1 (игра против компьютера) 
или 2 (игра против другого человека). Игрок против компьютера 
всегда играет справа. После выбора типа игры появится поле и будет пауза 
длительностю 3 секунды. Игра заканчивается когда одна из сторон пропустит 12-тый мяч

------УПРАВЛЕНИЕ В ИГРЕ------
Синий игрок:
    вверх - стрелка ?
    вниз - стрелка ?
Красный игрок:
    вверх - клавиша W
    вниз - клавиша S
Пауза (останавливается только мяч) - Пробел
Снять паузу - P (англ)
Закрыть игру - Esc

------Фишки------
Определенной фишкой есть возможность запустить игру с другим
человеком и во время игры нажать 1, тем самым вместо левого игрока 
(красный) будет играть компьютер. Также если во время игры с компьютером 
нажать 2, то управление над синим игроком перейдет человеку.
